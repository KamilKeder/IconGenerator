Informacje o aplikacji

O aplikacji
Aplikacja umożliwia generowanie ikon, z wybranymi przez użytkownika parametrami. Maksymalna wielkość to 300px x 300px. Użytkownik ma również możliwość pobrać wygenerowaną ikonę. Aplikacja korzysta z API icons8 oraz z filesaver.js.
Technologie
HTML, CSS, JavaScript, API, Bootstrap
